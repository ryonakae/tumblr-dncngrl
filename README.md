# tumblr-dncngrl
**Tumblr theme of Dancing GIrl.**

* Gulp
* React
* Webpack
* Jade
* Stylus


## Require
* Node.js


## Setup
### Clone repository

    $ git clone https://github.com/ryonakae/tumblr-dncngrl.git
    $ cd tumblr-dncngrl
    
### Install npm packages

    $ npm i
    
    
## Development
### Watch

    $ gulp watch
    
### Build

    $ gulp build
